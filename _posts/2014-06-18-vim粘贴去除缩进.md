vim在插入模式下，因为在.vimrc文件中设定了smartindent，所以粘贴过来的文本会自动进行缩进，因此，在进行粘贴之前，输入：

{% highlight vim %}
:set paste
{% endhighlight %}

粘贴之后，输入：
{% highlight vim%}
:set nopaste
{% endhighlight%}
